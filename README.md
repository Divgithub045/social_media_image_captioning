# CaptionCraft

AI-powered social media caption generator using a multimodal Transformer-based architecture.

---

## Overview

CaptionCraft generates context-aware and stylistically appropriate captions from images by combining visual features with a fine-tuned language model.

## Features

- Image-based caption generation  
- Multimodal vision–language fusion  
- Mood and aesthetic conditioning  
- LLM-based caption refinement

## Tech Stack

- Python  
- TensorFlow  
- Transformers
